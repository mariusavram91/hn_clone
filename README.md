# Hacker News Clone

Hacker News Clone built using Silex PHP micro framework + Vanilla Javascript. [See demo](http://hn-clone-2.herokuapp.com/)

Install requirements:
```
$ composer install
```

Run a server locally:
```
$ php -S localhost:8080 -t public
```
